    1. 之前需要安装 node 及 npm,目前是 node 自带 npm，下载 node
    2. 在新建文件夹中打开命令行工具（npm需要管理员权限）

    安装开始

    1. 全局安装 vue-cli，运行$ cnpm install –global vue-cli （只需要第一次运行）
    2. 创建一个基于 webpack 模板的新项目，运行 $ vue init webpack my-project（可能会出现比较慢的情况，多等会）
    3. 最后在项目所在文件夹运行 npm run dev，即可在浏览器运行

    第二步需要进行一些配置，默认回车即可

    遇到问题：

    1.windows 里面会出现npm无法下载到 vue-cli 的情况

    两个原因：

    1. 网络不好，使用淘宝 NPM 镜像。$ npm install -g cnpm –registry=https://registry.npm.taobao.org
    2. npm 没有权限（类似 ERROR:operation not permitted）,右键“开始”菜单，选择管理员运行，再次尝试
    2.chromedriver 下载失败，无法自动启动chrome浏览器，如下图：
